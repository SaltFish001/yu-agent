# Product

## Register

product

## Users

开发者（主要是项目作者本人）在桌面浏览器里与 yu-agent 编程 agent 对话、监控 topic 任务线程、查看状态、开终端。中文为主要界面语言。使用场景：长时间编码会话中的协作伙伴控制台。

## Product Purpose

yu-agent Web 控制台：让开发者以对话方式驱动 DeepSeek 编程 agent，并随时观察其思考过程、任务状态与系统健康。成功 = 用户感到 agent 是一位可靠、透明、反应清晰的同事，而不是一个黑盒。

## Brand Personality

沉静、可靠、有温度的工具。三词：**深水的静**、**暖灯的亲**、**仪表的准**。yu（鱼）生于水——界面气质取自"深水 + 暖灯"：冷调深水底色给人专注与纵深，一点暖琥珀光给出友好与生气。

## Anti-references

- 不要通用 SaaS 仪表盘（紫蓝渐变、玻璃拟态、hero 大数字）
- 不要 AI 默认三件套：奶白米色底 + 陶土橙、纯黑 + 酸性绿、报纸栅格
- 不要冷冰冰的监控面板感（Grafana 式密集图表墙）
- 不要过度圆润可爱的消费级聊天 App（我们是工具，不是玩具）

## Design Principles

1. **深水暖灯**：冷色深水底 + 单一暖琥珀信号色。暖色只用于"活"的东西——正在运行、连接正常、可行动作。
2. **状态即光**：活跃状态用柔和光晕表达，不用高饱和大色块。暗处一点光，而非满屏霓虹。
3. **工具消隐**：界面退到任务后面。一致的组件词汇、克制的装饰、可预期的交互。
4. **对话是主角**：聊天区域拥有最好的排版、最充分的留白、最柔和的形状。
5. **中文优先**：所有微文案以自然、亲和的中文书写；错误信息说清楚发生了什么、怎么办。

## Accessibility & Inclusion

- 暗/亮双主题，正文对比度 ≥ 4.5:1
- 尊重 `prefers-reduced-motion`
- 全键盘可达：focus-visible 清晰、命令面板、快捷键
- 状态不只依赖颜色（圆点 + 文案双编码）
